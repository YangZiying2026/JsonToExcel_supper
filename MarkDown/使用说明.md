# JsonToExcel 成绩分析转换器使用指南

欢迎使用 JsonToExcel 成绩分析转换器！本工具专为教育工作者设计，能够将教务系统导出的 JSON 格式原始成绩单，一键转换为包含多维分析、自动排名和分班统计的专业 Excel 报表。

## 🚀 核心功能

1.  **一键转换**：无需复杂操作，上传 JSON 文件即可自动生成报表。
2.  **多维排名**：
    *   **全年级总分排名**：包含原始总分和赋分总分（如适用）。
    *   **单科排名**：自动生成各学科的年级排名。
    *   **班级内部排名**：生成每个班级的独立报表，包含班级排名和年级排名对照。
3.  **智能赋分**：自动识别“政治、地理、化学、生物”等学科并进行赋分计算（支持自动跳过）。
4.  **深度统计**：提供平均分、中位数、最高/最低分及高分/低分名单统计。
5.  **隐私保护**：所有计算均在本地浏览器完成，数据不上传服务器。

## 📂 准备工作

在使用前，请确保您有以下文件：
1.  **原始成绩单 (JSON)**：必须包含学生姓名、班级、考号以及各科成绩。
2.  **(可选) 知识库 (Excel)**：用于修正或补充学生信息（如姓名、班级）。
3.  **(可选) 水印图片**：用于生成带水印的 Excel 报表。

## 🛠️ 操作步骤

### 第一步：上传文件
在主界面点击 **“上传 JSON 成绩单”** 区域，选择您的 JSON 文件。
*   *提示*：文件必须是 `.json` 格式。

### 第二步：高级设置 (可选)
*   **上传知识库**：点击右侧的“上传知识库”按钮，上传包含学生信息的 Excel 文件。
*   **上传水印**：点击“上传水印”按钮，选择一张图片作为报表背景。
*   **自定义文件名**：在输入框中输入您希望保存的文件名（无需后缀）。

### 第三步：生成报表
点击底部的 **“立即生成多维分析报表”** 按钮。
*   系统将自动进行数据清洗、语义分析、赋分计算和报表生成。
*   完成后，浏览器会自动下载生成的 `.xlsx` 文件。

## 💡 常见问题

**Q: 为什么显示“无效的 JSON 文件”？**
A: 请确保 JSON 文件内容是一个对象数组，且包含必要的字段（如姓名、班级）。

**Q: 如何跳过赋分计算？**
A: 只需将 JSON 文件名中包含“**赋分**”二字（例如 `高一期末赋分成绩.json`），系统即会自动跳过赋分步骤。

**Q: 生成的 Excel 打不开？**
A: 请确保使用 Microsoft Excel 2016+ 或 WPS Office 打开文件。
